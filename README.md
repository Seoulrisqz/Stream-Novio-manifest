## 📦 What is this?

This repository hosts a lightweight `addons.json` manifest list that can be used for quick Stremio addon sharing, copying, and loading.

It keeps multiple addon manifest links in one clean JSON file so they can be accessed from a single raw URL.

## 🚀 Usage

Copy the raw manifest link above and paste it into any compatible Stremio tool, loader, or addon installer that supports manifest lists.
