# Stremio Addon Repository

This repository contains a list of Stremio addons in a simple manifest format.

## Raw Manifest Link
https://raw.githubusercontent.com/Seoulrisqz/Stream-Novio-manifest/main/addons.json

## Usage
Paste the raw link into any compatible Stremio tool or loader to install all addons at once.

## Included Addons
- PCT Best4Stremio
- Superflix
- EasyNews
- RARBG Best4Stremio
